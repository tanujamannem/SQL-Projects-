# SQL Practice & Interview Preparation

This repository contains a comprehensive collection of SQL queries and concepts for learning. Each file is organized by topic and day, covering theory, examples, practice exercises, and challenge queries. The final file consolidates practical queries with answers for interview readiness.

---

## Repository Structure

| File | Description |
|------|-------------|
| `SQL_Interview_Preparation.sql` | Contains theory explanations and practical queries with answers for interview preparation. |
| `queries_day01_select_where.sql` | Introduction to `SELECT` and `WHERE` clauses with examples and practice queries. |
| `queries_day02_orderby_top.sql` | Sorting data using `ORDER BY` and retrieving top records with `TOP` or `LIMIT`. |
| `queries_day03_groupby_having.sql` | Aggregating data with `GROUP BY` and filtering with `HAVING`. |
| `queries_day04_joins.sql` | Different types of joins (`INNER`, `LEFT`, `RIGHT`, `FULL OUTER`) with examples and practice queries. |
| `queries_day05_subqueries.sql` | Using subqueries (single-row, multi-row, correlated subqueries) for complex queries. |
| `queries_day06_cte.sql` | Introduction to Common Table Expressions (CTEs) with practical examples. |
| `queries_day07_window_functions.sql` | Using window functions (`ROW_NUMBER`, `RANK`, `LEAD`, `LAG`) for advanced data analysis. |
| `queries_day08_joins_deepdive.sql` | Advanced join techniques and real-world query scenarios. |
| `queries_day09_union_intersect_except.sql` | Set operations: `UNION`, `INTERSECT`, and `EXCEPT`. |
| `queries_day10_window_advanced.sql` | Advanced window functions with partitioning and ordering. |
| `queries_day11_temptables.sql` | Temporary tables and their use cases for intermediate queries. |
| `queries_day12_derived_tables.sql` | Derived tables (subqueries in `FROM` clause) for modular queries. |
| `queries_day13_views.sql` | Creating and using views for reusable query structures. |
| `queries_day14_storedprocedures&functions.sql` | Writing stored procedures and functions with examples. |
| `queries_day15_Triggers.sql` | Introduction to triggers for automated actions on tables. |
| `queries_day16_Indexes.sql` | Indexing strategies to improve query performance. |
| `queries_day17_Transactions&Constraints.sql` | Transactions, ACID properties, and constraints for data integrity. |
| `queries_day18_Security&Grants.sql` | Database security, user roles, and granting permissions. |

---

## How to Use

1. Start from **Day 01** and progress sequentially to build your SQL knowledge.
2. Each file contains:
   - **Theory**: Key concepts and syntax.
   - **Examples**: Sample queries to understand the concept.
   - **Practice Queries**: Exercises for hands-on learning.
   - **Challenge Queries**: Advanced questions to test understanding.
3. **SQL_Interview_Preparation.sql**: Use this file for revision and interview practice with ready answers.

---

## Notes

- Recommended to execute queries on a sample database like **MySQLWorkBench**, or **any practice database**.
- Modify queries to match your database schema when necessary.
- Combine knowledge from multiple files to tackle complex real-world SQL problems.
